# MacOS Initial setup

## Brew
Install brew [Homebrew](https://brew.sh)

### Brew CLI applications
- [ ] python@2
- [ ] python@3
- [ ] yarn
- [ ] node@10

```sh
brew install python@2 python@3 yarn node@10
```

### Brew UI applications
- [ ] Sublime Text
- [ ] VSCode

```sh
brew cask install sublime-text visual-studio-code
```

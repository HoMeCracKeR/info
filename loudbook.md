# LoudBook API


## Methods

### Filse
```
# List of files
 > https://storage.loudbook.ru/storages8/{COURSE_ID}/recordsList.plist
 

# Intro
 > https://storage.loudbook.ru/storages8/{COURSE_ID}/records/000.mp3
 
# First lesson
 > https://storage.loudbook.ru/storages8/{COURSE_ID}/records/01.mp3
 
# Example:
 > https://storage.loudbook.ru/storages8/9079/records/01.mp3
```

### Categories
```
# Hits
 > https://api.dm-development.net/audiobooks/lb00/hits.json

# English
 > https://api.dm-development.net/audiobooks/lb00/learnenglish.json
```
